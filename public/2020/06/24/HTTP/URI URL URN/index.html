HTTP/URI URL URN

6月 24 评论

URI

Uniform Resource Identifier/ 统一资源标志符

用来唯一标识互联网上的信息资源

包括URL和URN

URL

Uniform Resource Locator /统一资源定位器

http://user:pass@host.com:80/path?query=string#hash

组成：

协议 域名 端口 资源路径 额外参数 锚点

Protocol:// Domain Name Port Path to the file Parameters Anchor

URN

永久统一资源定位符

在资源移动之后还能找到

目前没有成熟的使用方案
