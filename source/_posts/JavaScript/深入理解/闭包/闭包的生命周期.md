### 产生

在嵌套内部函数定义执行完成时就产生了（也就是产生指向上下文对象的时候，不是调用）

- `function`定义函数时，则是函数定义时就产生

- `var`定义的时候，则是代码执行时候产生

### 死亡

在嵌套的内部函数成为垃圾对象的时候



``` javascript
function fn1(){
  // 此时闭包已产生（函数提升，内部函数已经创建）
  var a = 2
  function fn2(){
    a++ 
    console.log(a)
  }
  return fn2
}

var f = fn1
f() // 3
f() // 4

f = null // 闭包死亡，f的重新指向导致包含闭包的函数成为了垃圾对象
```
