### 变量声明提升

- 通过var定义/声明的变量，在定义语句之前就可以访问到

- 默认初始值：`undefined`

``` javascript
var a = 1

function fn(){
  console.log(a)
  var a = 4 
}

fn() // undefined
```

### 函数声明提升

- 通过function声明的函数，在之前就可以调用
- 值：函数定义（对象）

``` javascript
console.log(b) // undefined
var b = 3
fn() //  fn 函数声明提升

function fn(){
  console.log('fn')
}

fn2() // TypeError, var 方式是变量声明提升
var fn2 = function(){
  console.log('fn2') 
}
```
