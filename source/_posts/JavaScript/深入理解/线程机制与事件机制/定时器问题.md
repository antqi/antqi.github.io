1. 定时器真的定时执行吗？

   - 不一定是真正的定时执行

     会先执行同步任务以及微任务，而定时器是宏任务

   - 一般会延迟一丁点（可以接受）

     ``` javascript
     console.time('----timer----')
     var timer = setTimeout(() => {
       clearTimeout(timer)
       timer = null
     
       console.timeEnd('----timer----')
     }, 200)
     // 一般在2000-205秒之间
     ```

   - 也可能延迟很长时间（无法接受）

      ``` javascript
     console.time('----timer----')
     var timer = setTimeout(() => {
       clearTimeout(timer)
       timer = null
     
       console.timeEnd('----timer----')
     }, 200)
     
     for (let index = 0; index < 1000000000; index++) {}
     // 500+ 毫秒后才执行定时器的回调函数
      ```

     

2. 定时器回调函数是哪个线程执行的

   主线程执行，JS引擎

3. 定时器是如何实现的
